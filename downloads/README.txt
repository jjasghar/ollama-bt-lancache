Ollama BitTorrent Lancache - Downloads Directory

This directory contains additional files that can be shared through the web interface.

Supported file types:
- .zip - Archive files
- .dmg - macOS disk images
- .exe - Windows executables
- .deb - Debian packages
- .rpm - Red Hat packages
- .tar.gz - Compressed archives
- .pdf - Documentation
- .txt - Text files

To add files:
1. Copy your files to this downloads/ directory
2. Refresh the web interface
3. Files will appear in the downloads section

Security note: Only place trusted files in this directory as they will be publicly accessible.
